# Salfx-web-project

## Project 2 - HTML Foundation Page

This project is a simple personal website created using only HTML. The website introduces me and provides some information about my interests and hobbies such as gaming, fitness, and cooking. The site contains two pages: a homepage and an About Me page that includes an image and a table showing my weekly activities.

While creating this project, I was able to learn how to structure a webpage using basic HTML elements such as headings, paragraphs, lists, images, tables, and navigation links. I also was practicing using inline HTML styling to change colors, font sizes, spacing, and page backgrounds. This was my first time messing and adding colors to an assignment. Most of the time it was usually some code and that's it.

One challenge I faced was organizing the folder structure correctly and making sure that the image path worked properly between the root folder and the html folder. After adjusting the file path, the image loaded correctly on the webpage. Another challenge that I had while working on this project was remembering how the colors work, since this was my first time actually implementing them.
